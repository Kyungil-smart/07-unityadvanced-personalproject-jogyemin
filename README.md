## 프로젝트 개요

- 게임 제목: Forgotten Dungeon : The Eight Trials

- 게임 장르: 3D 공포 / 이상현상 탐지 게임

- 개발 엔진: Unity6 (3D 6000.3.91f1)

- 개발 기간: 2026.02.26 ~ 2026.03.05

## 게임 소개
3D 공포 기반의 이상현상 탐지 게임, 플레이어는 정체불명의 폐성 내부에 갇히게 되며,\
반복되는 맵을 탐색하면서 이상현상(Anomaly)을 찾아야한다

게임은 스테이지 구조로 진행된다.
각 스테이지에서 플레이어는 주변 환경을 관찰하고\
이상현상이 존재하는지 여부를 판단한 뒤 올바른 출구를 선택해야 한다.

- 이상현상이 있다고 판단하면 후문
- 이상현상이 없다고 판단하면 정문

## 조작 방법

|키|기능|
|---|---|
|W/A/S/D|플레이어 이동|
|마우스|시점 회전|
|Shift|달리기|
|G|문 상호작용 / 스테이지 진행|
